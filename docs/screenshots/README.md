# Screenshot Guide

Use this folder for GitHub, marketplace, and documentation screenshots.

## Recommended Captures

- `completion-message.png`: a clean advancement completion message in chat.
- `progress-message.png`: a multi-criterion advancement progress message.
- `hover-details.png`: hover text showing title, description, key, and progress.
- `config-highlight.png`: a cropped view of the broadcast or format config.

## Capture Notes

- Use a readable Minecraft GUI scale.
- Avoid full-screen console logs unless they show a specific admin command result.
- Redact player IPs, private server names, and private datapack keys if needed.
- Keep images at least 1280px wide when possible.
